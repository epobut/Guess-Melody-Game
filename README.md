# 🎵 Guess-Melody-Game

[UA] Класична вікторина «Вгадай мелодію» на C# (WinForms).  
[EN] A classic "Guess the Melody" quiz game built with C# (WinForms).

---

## ⚡ Особливості / Features

### 🇺🇦 Українською:
- **Музична вікторина:** Вгадуйте пісні за короткими фрагментами.
- **Мультиплеєр:** Підтримка гри для двох гравців із системою підрахунку балів.
- **Налаштування:** Зручний інтерфейс для керування плейлистами та тривалістю раундів.
- **Двигун:** Використовує нативні компоненти Windows для стабільного відтворення аудіо.

---

### 🇺🇸 English:
- **Music Quiz:** Guess songs after hearing short audio clips.
- **Multiplayer:** Battle your friends with a built-in scoring system.
- **Customizable:** Manage your own playlists and adjust round settings easily.
- **Native Audio:** Powered by Windows Media components for reliable performance.

---

## 🚀 Як запустити / How to run

1. Відкрийте файл рішення `.sln` у **Visual Studio**.
2. Відновіть залежності та зберіть (Build) проект.
3. Додайте музичні треки в налаштуваннях програми або вкажіть шлях до папки з музикою.
4. Натисніть **F5 (Start)** та починайте гру!

---

## 🛠 Технології / Tech Stack
- **Language:** C#
- **Framework:** .NET Framework / WinForms
- **Library:** WMPLib (Windows Media Player component)
